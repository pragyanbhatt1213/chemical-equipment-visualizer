# Chemical Equipment Visualizer

A comprehensive data analysis platform for chemical engineers to upload, analyze, and visualize equipment datasets. The system processes CSV files containing chemical equipment operational parameters (flowrate, pressure, temperature) and generates detailed analytics including:

- Equipment type distribution analysis
- Health scoring and outlier detection
- Efficiency rankings and risk assessments
- Statistical summaries (min, max, median, standard deviation)
- Professional report generation in PDF, CSV, and Excel formats

The platform serves chemical engineers who need to monitor equipment performance, identify operational issues, and generate documentation for compliance and optimization purposes.

## Key Features

- **Multi-platform access**: Web application and desktop PyQt5 client
- **Secure authentication**: Token-based authentication system
- **Advanced analytics**: Health scoring, outlier detection, efficiency rankings
- **Export capabilities**: PDF reports, CSV summaries, Excel spreadsheets
- **Data visualization**: Interactive charts and dashboards
- **Upload history**: Track and manage previous analyses
